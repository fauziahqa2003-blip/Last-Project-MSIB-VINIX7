# Last-Project-MSIB-VINIX7
# University Program Clustering using K-Means

**Overview**
This project was developed during the MSIB Independent Study Program (Junior Data Scientist) at VINIX7.
The objective of this project is to classify university study programs based on admission capacity and tuition fees using the K-Means Clustering algorithm.
The project includes data preprocessing, clustering analysis, evaluation using Elbow Method and Silhouette Score, and interactive dashboard visualization.
